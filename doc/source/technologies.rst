Technologies & Stack
====================

This page lists all technologies, frameworks, and tools used in the project.

Backend
-------

- **Django 3.0**: Web framework
- **Python 3.10**: Programming language
- **SQLite**: Database (development & production)
- **gunicorn**: WSGI HTTP server (production)
- **WhiteNoise 6.11.0**: Static files serving

Testing & Quality
-----------------

- **pytest**: Testing framework
- **pytest-cov**: Code coverage measurement
- **flake8**: Code linting (PEP8 compliance)

CI/CD & Deployment
------------------

- **GitHub Actions**: Continuous integration pipeline
- **Docker**: Containerization
- **DockerHub**: Container registry
- **Render**: Production hosting platform

Monitoring
----------

- **Sentry**: Error tracking and monitoring

  - ``send_default_pii=True``: User context capture
  - ``enable_logs=True``: Application logs integration

Documentation
-------------

- **Sphinx**: Documentation generator
- **Read the Docs**: Documentation hosting
- **sphinx-rtd-theme**: Documentation theme
- **sphinx-autodoc-typehints**: Type hints in documentation

Frontend
--------

- **HTML5/CSS3**: Templates and styling
- **JavaScript**: Client-side interactions
